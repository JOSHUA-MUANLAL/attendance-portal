# Attendance Portal

## How to install :-

* Clone the repository
  ```
  git clone https://github.com/sanjaybora04/attendance-portal
  ```
* Install Packages
  ```
  npm i
  ```
  

* ### Configuration :-

* create a `.env` file in root folder and enter this
  ```
  VITE_SERVER_URL=http://localhost:5000/
  ```

## How to Run :-

* Run this command in root folder
```
npm run dev
```
